세브란스병원

PROJECT FEATURE 기업 데스크탑 & 모바일 페이지
RELEASE DATE 2022.02

연세대학교 부속 병원인 세브란스병원의 웹사이트를 리뉴얼하였다.
이미지 중심이었던 기존의 사이트와 달리 직관적이고 정확한 정보를 전달하는 것에 중점을 두었다.
기존의 불필요한 요소들을 삭제하고,
정보 전달을 중심으로 리뉴얼해 사용자로 하여금 더욱 쉽게 다가갈 수 있도록 하였다.
전략 수립
홈페이지 전면개편

: 세브란스병원 홈페이지 개편을 통한
고객 인지도 향상


효과적인 마케팅 전략 수립

: 세브란스병원의 이미지 향상
및 홍보로 비즈니스 창출


핵심 콘텐츠 강화

: 빠른 정보 전달을 위한
홈페이지 제작 및 서비스 구현


효율적인 UI 구현

: 최적화된 User Interface 구현으로
잠재적 고객 유치


CRM 강화

: 체계적이고 지속적인
사용자의 분석을 통한 맞춤 정보 제공
